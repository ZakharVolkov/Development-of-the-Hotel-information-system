Для запуска приложения потребуется Visual Studio 2022, а также СУБД MySQL Workbench 8.0 для того, чтобы не было конфликта между версиями.

Запуск приложения:
1. Открыть СУБД MySQL Workbench 8.0 CE, далее, выбрать в MySQL Connections – "Local instance MySQL80".
2. Открыть папку с файлам дампов БД все файлы перенести в открытое пространство MySQL.
3. Выполнить инструкцию всех дампов.
4. Далее, зайти запустить файл DM Volkov.sln
После загрузки проекта в коде потребуется изменить логин и пароль БД на Ваш, для этого надо:
1) Открыть исходный код через «Обозреватель решений».
2) В 17 строчке кода поменять пароль и логин на Ваш.
5. Запускаем программу. 

Для входа как:
1. "Администратор" пароль: admin
2. "Портье" пароль: porter

To run the application, you will need Visual Studio 2022, as well as the MySQL Workbench 8.0 DBMS so that there is no conflict between versions.

Launching the application:
1. Open the MySQL Workbench 8.0 CE DBMS, then select "Local instance MySQL80" in MySQL Connections.
2. Open the folder with the database dump files, transfer all files to the open MySQL space.
3. Follow the instructions of all dumps.
4. Next, go to run the DM Volkov.sln file
After downloading the project in the code, you will need to change the login and password of the database to yours, for this you need:
1) Open the source code through the "Solution Explorer".
2) In line 17 of the code, change the password and login to yours.
5. Launch the program.

To log in as:
1. "Admin" password: admin
2. "Porter" password: porter
